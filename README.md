Название: Лендинг ALIVIO

Стек: HTML, CSS, JS

Задачи:
1. Разработал адаптивную верстку по подходу Desktop-First.
2. Организовал архитектуру стилей с помощью блоков, что упростило поддержку кода.
3. Настроил валидную, семантическую верстку для улучшения SEO-показателей сайта.

Фичи: 
1. Реализовал кастомную логику управления видеоплеером с помощью JS, отслеживая состояния воспроизведения и паузы, а также движения мыши.
2. Реализовал архитектуру переключения состояний меню на базе класса-модификатора. Это позволило вынести всю визуальную логику в CSS, оставив JS-скрипт максимально легковесным и чистым.
